4. Using the “glm()” function, perform multiple logistic regression. Ensure that the habitat types (i.e., HabitatCode) are treated as categorical variables by converting them to factors.

mousedeerlm <- mousedeerclean[, c("AverageDistance", "Effort", "HabitatCode", "Detected")]
mousedeerlm$HabitatCode <- as.factor(mousedeerlm$HabitatCode)
x1 <- mousedeerlm$AverageDistance
x2 <- mousedeerlm$Effort
x3 <- mousedeerlm$HabitatCode
y <- mousedeerlm$Detected
data <- data.frame(x1, x2, x3, y)
model <- glm(y ~ x1 + x2 + x3,
  data = data,
  family = binomial)
data$Probability <- predict(model, type = "response")
data$Prediction <- ifelse(data$Probability >= 0.5, 1, 0)
print(data)

5. After fitting the model, create a logistic regression curve plot for each independent variable used. Export the plots as PDF files.

Average Distance vs Probability of Detection

new_x1 <- data.frame(
  x1 = seq(min(data$x1), max(data$x1), length.out = 19),
  x2 = mean(data$x2),
  x3 = levels(data$x3)[1]
)
new_x1$prob <- predict(model, newdata = new_x1, type = "response")
ggplot()+
